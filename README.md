# RedLine – Bus Booking System

RedLine is a bus booking and transport management system that allows users to view available buses and make bookings through a simple and interactive interface.

---

## Features
- View available buses  
- Book tickets through a user-friendly interface  
- Manage bus and booking data  
- Interactive and responsive frontend  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, C++  
- Database: JSON (file-based storage)
